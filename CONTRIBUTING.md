# 参与贡献

感谢你帮助改进这份大模型面试题图解资料。

## 内容勘误

题目、答案或图解有事实性问题时，请优先创建“内容勘误” Issue，并提供：

1. 题目 ID、模块链接或 SVG 文件名；
2. 需要修正的原句或图中节点；
3. 论文、官方文档、源码或可复现实验；
4. 建议修改方式，以及是否会影响官网正式答案。

生成内容来自维护者的上游题库。为避免下一次自动同步覆盖修改，请不要直接向
`README.md`、`modules/`、`images/` 或 `review-manifest.json` 提交内容 PR。维护者会先在上游修正，
通过复核与哈希检查后再同步到本仓库。

## 可以直接提交的改动

- 勘误流程、Issue 模板等仓库治理文件；
- 不改变题目结论的可访问性与阅读体验修复；
- 能够独立复现的检查脚本或文档改进建议。

提交 PR 时，请说明问题、修改范围、验证方式和资料来源。一个 PR 尽量只解决一个问题。

## 许可说明

在根目录 `LICENSE` 落地前，仓库内容默认保留全部权利。提交内容即表示你有权提供该内容，
但不代表维护者已经接受或发布该贡献。
