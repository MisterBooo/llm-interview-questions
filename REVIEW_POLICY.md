# 内容复核办法

本仓库的目标是让图解、30 秒回答与站内正式答案保持一致，不以题目数量替代内容可信度。

## 发布前检查

1. 每道题必须维护独立的陈述句摘要，不能复用用于引出追问的字段。
2. 每道题记录正式答案 SHA-256；答案变化后必须重新核对摘要和图解。
3. 每张 SVG 记录 SHA-256；图解变化后必须重新复核。
4. 生成器检查题目、slug、复核记录是否重复，并拒绝不安全 SVG 内容。
5. B 级条目保留源题中的可回查链接；C 级条目标明为教学整理。

本轮记录中的 `reviewedBy: codex-content-audit` 表示使用 Codex 逐题完成了图文一致性初审，不冒充人工专家签名。首次公开发布已完成最终抽查与许可确认；公开后仍按本办法持续抽查并处理勘误，当前许可状态以根目录 `LICENSE` 为准。

## 不做的承诺

- 不把“有图”写成“全网最值得学”。
- 不把教学整理写成已经核验的公司真题。
- 不承诺某个算法、框架或参数在所有硬件与业务场景下都更优。
- 不把同一 README 中的多个链接描述成多个独立站点背书。
